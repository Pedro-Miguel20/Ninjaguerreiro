## 📝 Project Ninjaguerreiro

#### 📜 Formulário em HTML usando MySQL database usando PHP.

### Versão 1.0 (PHP) ✔️
Nesta versão se encontra o gabiru.PHP que apenas armazena as informações dentro das variáveis e printa elas na tela seguinte.
